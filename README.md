# 🏡 Room Management Backend

## 📖 Introduction
This is the backend of an **Room Management System**, built using **Spring Boot** and **MySQL**.

## 🚀 Technologies Used
- **Java 21**
- **Spring Boot 3.4.4**
- **MySQL**
- **JWT Authentication**
